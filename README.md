### 经济周期与股票价格的关系研究

基于经济周期对投资者的过度自信等心理进行分析，建立VAR模型，分析脉冲函数识别内生变量之间的关系。

#### 文件目录

1. data:存放过程中的数据
2. paper: 参考文献及论文
3. pics：保存的一些重要图片
4. src：源码
